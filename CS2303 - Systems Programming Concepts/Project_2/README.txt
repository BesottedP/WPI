This program fetches info from serveral data files and compiles them into comprehendable text for the user. The user has
multiple different options on how the data is presented.

To compile the program, run the command:
    gcc -g -Wall register.o coffee_shop.o -lm -o register

To run the program, run the command:
    ./register
then answer the printed prompt
