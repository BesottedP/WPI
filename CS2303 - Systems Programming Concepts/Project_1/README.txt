This program simulates a race between 3 animals with varying movements. The movements are randomly chosen using a seed provided by the user.

To compile the program, run the command:
    gcc -g -Wall race.o animals.o -lm -o theRace

To run the program, run the command:
    ./theRace
then enter a number once prompted
